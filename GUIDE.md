## 행사 당일 (D-Day)

### 준비물 체크리스트
- [ ] 관리자용 노트북/태블릿 (프로젝터 연결)
- [ ] 프로젝터/대형 스크린
- [ ] 인쇄된 QR 코드 (백업용)
- [ ] 안정적인 인터넷 연결 (와이파이 또는 핫스팟)

### 행사 시작 30분 전

1. **기기 세팅**
   - 관리자 노트북을 프로젝터에 연결
   - 관리자 페이지 접속: `https://your-event.vercel.app/admin`
   - 비밀번호로 로그인

2. **투표 생성**
   - 질문 입력
   - 선택지 입력 (2~6개)
   - "투표 후 참가자에게 결과 표시" 체크 (선택)
   - "투표 생성하기" 클릭

3. **QR 코드 배치**
   - 행사장 입구에 QR 코드 포스터 부착
   - PPT 첫 슬라이드에 QR 코드 삽입
   - 스크린에 QR 코드 표시

### 투표 진행

1. **투표 시작**
   ```
   사회자: "화면의 QR 코드로 접속해주세요!"
   관리자: [투표 시작] 버튼 클릭
   ```

2. **실시간 모니터링**
   - 화면에 실시간 차트 표시
   - 총 투표 수 확인
   - 득표율 변화 관찰

3. **참여 독려**
   ```
   사회자: "현재 XX명이 투표하셨습니다!"
   사회자: "아직 안하신 분들은 서둘러주세요!"
   ```

4. **투표 종료**
   ```
   사회자: "투표를 마감하겠습니다!"
   관리자: [투표 중지] 버튼 클릭
   ```

5. **결과 발표**
   - 화면에 표시된 차트 활용
   - 각 선택지별 득표수와 비율 설명
   - 우승 항목 발표

### 추가 투표 진행

같은 행사에서 여러 투표를 진행하는 경우:

1. [투표 초기화] 버튼 클릭 (이전 데이터 삭제)
2. 새로운 질문과 선택지 입력
3. [투표 생성하기] 클릭
4. [투표 시작] 클릭
5. 참가자들은 **새로고침 불필요** (자동 업데이트)

## 문제 해결

### 투표가 안 보여요
**참가자 화면:**
- "현재 진행 중인 투표가 없습니다" 표시
- 관리자가 투표를 생성했는지 확인
- 관리자가 [투표 시작] 버튼을 눌렀는지 확인
- 페이지 새로고침 (F5)

### 투표 후 "투표가 완료되었습니다" 화면이 안 나와요
- 정상입니다! 투표는 정상 처리됨
- 브라우저 세션 문제일 수 있음
- 다른 브라우저로 시도

### 중복 투표가 되어요
**현재 구현 방식:**
- 같은 브라우저 세션에서는 한 번만 투표 가능
- 브라우저 종료 후 재접속 시 다시 투표 가능
- 시크릿 모드에서는 투표 가능

**대응 방안:**
1. 참가자들에게 안내: "브라우저를 닫지 마세요"
2. 투표 시간을 짧게 설정 (1-2분)
3. 관리자가 비정상 투표 패턴 모니터링

### 인터넷이 불안정해요
**백업 플랜:**
1. 모바일 핫스팟 활성화
2. 관리자 화면 스크린샷 저장
3. 수동 투표 진행 (손들기)

### 너무 많은 사람이 동시 접속해요
Firebase 무료 플랜 제한:
- 동시 연결: 100개
- 읽기/쓰기: 충분함

100명 초과 시:
1. 분산 투표 (그룹별)
2. Firebase 유료 플랜 업그레이드

## 행사 후

### 1. 데이터 백업 (선택)
Firebase Console → Realtime Database → 데이터 내보내기 (JSON)

### 2. 데이터베이스 삭제 ⚠️
**중요: 테스트 모드는 보안 취약**

Firebase Console → Realtime Database → 설정 → "데이터베이스 삭제"

### 3. 프로젝트 정리 (선택)
Firebase Console → 프로젝트 설정 → "프로젝트 삭제"

## 팁 & 노하우

### 투표 질문 작성
✅ **좋은 예:**
- "가장 인상 깊었던 세션은?"
- "다음 행사에서 듣고 싶은 주제는?"
- "점심 메뉴 선택!"

❌ **피해야 할 예:**
- 너무 긴 질문 (2줄 이상)
- 모호한 질문
- 예/아니오로 답할 수 없는 질문

### 선택지 작성
- **2-4개 추천** (너무 많으면 혼란)
- **간결하게** (한 줄 이내)
- **명확하게** (중복되지 않게)

### 참여율 높이기
1. **사전 안내**: 행사 시작 전 투표 예고
2. **인센티브**: "투표하신 분 중 추첨으로..."
3. **실시간 공개**: 현재 투표 상황 화면 표시
4. **마감 임박 알림**: "1분 남았습니다!"

### 화면 구성 팁
- 관리자 페이지를 프로젝터 전체 화면으로
- 실시간 차트가 잘 보이도록 확대
- 어두운 조명에서도 잘 보이는지 확인
